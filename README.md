# diceware

Diceware is cool, but sometimes you don't need the security of a real RNG and a CSPRNG will suffice.

## Installation

    go get -u github.com/holizz/diceware

## Usage

    % diceware
    cud rhino clink ovary vent stoke lab eater
    % diceware -d=_ -c=5
    ascend_grim_shin_oy_muzo
